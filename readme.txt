Just scan with your phone and you'll automatically logged in.

== Description ==
The plugin will add the scan2login QR code to your login page.

When the user/admin scans the QR code (with the Scan2Login smartphone application), the login information will be filled out automatically and the user will be automatically logged in.

The plugin, let you and your users, use strong password without the need to memorized them.

The plugin may be used free of charge by non-profit organizations, educational institutions, and bloggers.

== Installation ==

1. Upload `scan2login.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
